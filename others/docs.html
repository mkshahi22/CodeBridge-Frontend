Overview

CodeBridge is an AI-powered platform designed to help developers build software faster through natural language-based code generation. From components to full applications, CodeBridge accelerates coding productivity.

What CodeBridge Offers

  • ChatGPT-style dashboard for AI code generation
  • Optimized, explainable, and debuggable code
  • File upload for context-aware output
  • History tracking with search filters
  • Community sharing for public snippets
  • User preferences & settings customization

Setup

No installation needed. Everything runs in the browser.

  • Frontend: HTML + Tailwind + JavaScript
  • Backend: Spring Boot + MySQL + JWT
  • AI Engine: Kiro API (Vibe Coding + MCP)

Authentication

Authentication uses JWT stored securely in localStorage. 

POST /auth/login
{
    "email": "user@example.com",
    "password": "******"
}

Code Generation API

This endpoint sends your prompt to Kiro AI for generation: 

POST /generate
Headers:
  Authorization: Bearer <token>

Body:
{
    "prompt": "Create a React login page",
    "files": []
}

File Upload

Supported file types:

  • .js, .py, .java, .html, .css
  • .txt, .md, .json
  • .pdf (text extract)

History

Every generation is saved automatically. You can view, copy, or delete past outputs.

Settings

Customize language, detail level, notifications, and privacy options.

FAQ

Is CodeBridge free?

Yes, completely free.

Which AI do you use?

We use Kiro AI with Vibe Coding + MCP.

Can I upload files?

Yes, multiple formats supported.

Key Concepts

  • Prompt: Natural language request for code.
  • Generation: Output created by AI.
  • Context Files: Uploaded support files.
  • Community Post: Publicly shared output.

Error Handling

{
    "status": 400,
    "error": "Invalid prompt",
    "message": "Prompt cannot be empty"
}

Most errors occur due to missing tokens or invalid input.

Best Practices

  • Use short and clear prompts.
  • Break problems into smaller tasks.
  • Upload files when context is required.
  • Review generated code manually.

Limitations

  • Some advanced frameworks may need tweaking.
  • Large uploads may slow down response.
  • AI output may not always be optimized.
